# Bag of Gifs

A dependency-free birthday site filled with local animated GIFs and a browser-generated birthday tune.

Open `index.html` directly in a browser, or serve the folder with any static web server.

## Files

- `index.html` - page structure
- `styles.css` - responsive party styling
- `script.js` - GIF randomizer, confetti, and Web Audio tune
- `Bag of gifs/` - local GIF assets

## GitHub Pages

This repo is static. Enable GitHub Pages for the default branch and the site will serve from the repository root.
